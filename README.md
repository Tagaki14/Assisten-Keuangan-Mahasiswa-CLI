# 💰 Assisten Keuangan Mahasiswa CLI

**Assisten Keuangan Mahasiswa CLI** adalah alat bantu baris perintah yang dirancang khusus untuk mahasiswa yang ingin mengelola keuangan mereka dengan efisien tanpa gangguan antarmuka grafis yang berat. Catat uang saku, pengeluaran kos, hingga biaya kuliah langsung dari terminal Anda.

## 🚀 Fitur Utama

- **💸 Catat Transaksi:** Masukkan pemasukan dan pengeluaran dengan cepat.
- **📊 Laporan Keuangan:** Lihat ringkasan saldo dan riwayat transaksi secara real-time.
- **🎯 Anggaran Bulanan:** Tetapkan batas budget dan dapatkan peringatan jika mendekati limit.
- **💾 Export Data:** Simpan data ke format CSV atau JSON untuk backup atau analisis lebih lanjut.
- **⚡ Ringan & Cepat:** Dibuat dengan [Sebutkan Bahasa Pemrograman, misal: Go/Python/Rust], berjalan lancar di semua sistem operasi.
